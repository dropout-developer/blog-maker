\```bash
npm install
\```

### Set Up Environment Variables
Create a `.env` file in the root directory.
Add your OpenAI API key:
\```makefile
OPEN_AI_API_KEY=your_openai_api_key
\```

### Run the Application
\```bash
ng serve
\```
Open your browser and navigate to http://localhost:4200/ to access the app.
